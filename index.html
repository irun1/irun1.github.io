How to download and install Infinity Run?

  1. Click on the Download button.
  2. After downloading, open File Explorer and navigate to the downloaded zip file.
  3. Right-click on the selected zip file.
  4. Select Extract All...
  5. Click Extract.
  6. Close File Explorer and open BlueJ.
  7. If you do not have BlueJ or want to update it, visit bluej.org.
  8. Click on Project.
  9. Select Open Project...
  10. Choose the Infinity Run folder from where you extracted the zip file.
  11. Click Select Folder.
  12. You have downloaded and installed Infinity Run on your device. Enjoy the game.

How to update Infinity Run?

  1. Check if there is a new version of Infinity Run available by comparing the version numbers from this website and the README file of your installed game.
  2. If there is a new version available, delete all files of Infinity Run from your device.
  3. Click on the Download button.
  4. After downloading, open File Explorer and navigate to the downloaded zip file.
  5. Right-click on the selected zip file.
  6. Select Extract All...
  7. Click Extract.
  8. Close File Explorer and open BlueJ.
  9. Start the game and click the UPDATE button to update the Infinity RUn file.
  10. You have successfully updated Infinity Run on your device. Enjoy the updated game.

How to add new profiles to the main game?

  1. To add your new profiles to the main game to showcase your high score or to get mentioned in the game, you can click on the Submit File button.
  2. Fill out the form.
  3. To upload the file, click the Upload File button in the form.
  4. Select the Infinity Run.txt file.
  5. Click Select and submit the form.
  6. Your new profiles will be added to the main game within a few hours.

How to give feedback or report any bugs?

  1. For any feedback or issue, you can contact me by clicking on the Submit File button.
